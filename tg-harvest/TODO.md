## Середній пріоритет

### Аналітика хештегів і згадок
- Топ хештегів, згадок, URL за частотою (дані вже є в entities)
- Хмара хештегів на сторінці аналітики
- Фільтр пошуку за хештегом

### Пакетний парсинг
- CLI: `tg-harvest parse --batch channels.txt` — список каналів з файлу
- Web UI: черга парсингу з прогресом для кожного каналу
- Автоматична затримка між каналами для уникнення FloodWait

### Markdown-експорт
- Новий `MarkdownExporter` — формат для Obsidian, Notion, GitHub
- Формат: `# Канал\n## Повідомлення [ID]\nДата: ...\nТекст: ...`

### Розширений пошук
- Regex-режим (перемикач у веб-інтерфейсі)
- Оператори: AND, OR, NOT
- Збережені пошукові запити

### Експорт статистики
- CLI: `tg-harvest export-stats output/ --format json`
- Вивід: time-series, агрегати, топ-N списки
- Формат для BI-інструментів (Tableau, Power BI, Grafana)
